= ojosama
:toc: left
:sectnums:

テキストを一般人お嬢様口調に変換します。

== 使い方

ワンライナーでテキストを変換する場合は、以下のように実行します。

[source,bash]
----
$ ojosama -t これは草です
これはおハーブですわ
----

テキストファイルをまるごと変換する場合は、以下のように実行します。

[source,bash]
----
$ ojosama README.adoc
----

== 変換アルゴリズム
