= ojosama
:toc: left
:sectnums:

テキストを一般人お嬢様口調に変換します。

== 使い方

=== コマンド

ワンライナーでテキストを変換する場合は、以下のように実行します。

[source,bash]
----
$ ojosama -t これは草です
これはおハーブですわ
----

テキストファイルをまるごと変換する場合は、以下のように実行します。

[source,bash]
----
$ ojosama README.adoc
----

=== ライブラリ

Goのコードとして使う場合は以下のように使用します。

[source,bash]
----
package main

import (
  "fmt"

  "github.com/jiro4989/ojosama"
)

func main() {
  s := "育ちは一般家庭です"
  text, err := ojosama.Convert(s, nil)
  if err != nil {
    panic(err)
  }
  fmt.Println(text)
}
----
